# Expense Tracker

## About

This is a full stack project using PostgreSQL, Express, React and Node.js.

## Prerequisites

Node.js, React.js

## Installation

Run `npm run dev` on the project root will launch both the server and the client side.

## Ports

-   Backend: `127.0.0.1:8080`
-   Frontend: `127.0.0.1:3000`
